# LinkedIn Pipeline Playbooks

Free, practical resources for B2B and SaaS founders and GTM teams building pipeline on LinkedIn: prompt packs, templates, and frameworks you can copy and use today.

Maintained by [PromiseClick](https://www.promiseclick.com). Browse the full library, with guides and worked examples, at **[promiseclick.com/resources/playbooks](https://www.promiseclick.com/resources/playbooks)**.

## What's inside

### Prompt packs
- **[Founder Hooks Prompt Pack](prompts/founder-hooks.md)** - a master AI prompt plus 12 proven hook formulas (each with an example) to write LinkedIn posts that stop the scroll.

### Claude skills
- **[LinkedIn DM Writer](claude-skills/linkedin-dm-writer/SKILL.md)** - an installable Claude skill that writes warm, personalized LinkedIn DMs and follow-up sequences (no pitch, high reply rate). Add it to Claude once and draft outreach in seconds.

_On the way: positioning templates, the content-to-pipeline framework, and warm outreach templates._

## How to use

1. Open the pack you want, for example [prompts/founder-hooks.md](prompts/founder-hooks.md).
2. Copy the master prompt into ChatGPT or Claude, fill in the bracketed inputs, and run it.
3. Or use the fill-in formulas directly. Each one comes with an example.

## Who it's for

B2B and SaaS founders and GTM teams who use LinkedIn to generate pipeline and want practical, copy-and-use resources instead of theory.
