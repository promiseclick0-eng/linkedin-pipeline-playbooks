# LinkedIn Character Limits and Formatter Reference

Last verified: July 22, 2026

LinkedIn changes product limits and sometimes shows different limits by plan or product surface. Check the counter in the current LinkedIn interface before publishing. This reference separates limits confirmed in LinkedIn Help from limits that should be verified in the live field.

## Confirmed limits

| LinkedIn field | Current maximum | Practical recommendation | Source |
|---|---:|---|---|
| Feed post | 3,000 characters | Use only the space the idea needs. Format for mobile before publishing. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a528176/) |
| Article | 125,000 characters | Structure long articles with clear sections and evidence. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a522483/differences-between-posting-updates-and-publishing?lang=en) |
| Basic connection note | 200 characters | Write one truthful reason to connect. Do not pitch. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a563153/personalize-invitations-to-connect?lang=en) |
| Premium connection note | Up to 300 characters on documented Premium surfaces | Check the live counter because LinkedIn Help pages currently describe different product experiences. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a6239760) |
| InMail subject | 200 characters | Use a short, specific subject instead of filling the available space. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a546814/purchasing-and-sending-an-inmail?lang=en) |
| InMail body | 2,000 characters | Keep the message focused and easy to answer. | [LinkedIn Help](https://www.linkedin.com/help/linkedin/answer/a546814/purchasing-and-sending-an-inmail?lang=en) |

## Common profile and engagement fields

These values can change without a matching LinkedIn Help update. Use them as working checks, then confirm against the counter shown in the live field:

| LinkedIn field | Working limit to verify | What matters most |
|---|---:|---|
| Personal profile headline | 220 characters | Put the buyer and outcome before titles and credentials. |
| Personal profile About section | 2,600 characters | Make the opening lines useful before the reader selects See more. |
| Comment | 1,000 characters | Add one useful thought. Turn a longer response into a post when appropriate. |

## Formatting workflow

1. Draft for clarity before counting characters.
2. Remove throat-clearing, repeated context, and filler.
3. Put the most useful information before any visible cutoff.
4. Use short paragraphs for mobile reading.
5. Confirm the final count in LinkedIn's current interface.
6. Preview posts with the [LinkedIn Post Formatter](https://www.promiseclick.com/linkedin-free-tools/linkedin-post-formatter).
7. Preview About copy with the [LinkedIn About Section Formatter](https://www.promiseclick.com/linkedin-free-tools/linkedin-about-section-formatter).
8. Build concise headline options with the [LinkedIn Headline Generator](https://www.promiseclick.com/linkedin-free-tools/linkedin-headline-generator).

## Connection-note warning

LinkedIn Help currently has conflicting documentation about how many personalized connection notes Basic accounts can send each month. Do not design a workflow around a fixed monthly allowance. Check the user's current account interface. The 200-character Basic mode is the safest default, and an expanded 300-character version can be supplied for Premium interfaces that show that limit.

## Copy checklist

- Count spaces and punctuation.
- Preserve the specific detail that makes the message credible.
- Do not trade clarity for using the full limit.
- Do not split one coherent idea across several comments or messages merely to bypass a limit.
- Recheck limits when LinkedIn changes the editor or publishing experience.
