# References

Source-linked working references for LinkedIn fields, limits, and publishing decisions.

## Available references

### [LinkedIn Character Limits and Formatter](linkedin-character-limits.md)

Use it to check current working limits for posts, articles, connection notes, InMail, headlines, About sections, and comments.

The reference distinguishes between:

- Limits confirmed in official LinkedIn Help
- Working limits that should be checked in the live LinkedIn field
- Practical recommendations that help readability but are not platform limits

Quick workflow:

1. Find the field you are writing.
2. Check the source and last-verified date.
3. Draft for clarity instead of trying to fill the maximum.
4. Confirm the final count in your current LinkedIn interface.
5. Use the linked PromiseClick formatter or generator when relevant.

Platform limits can change. If the live interface conflicts with this repository, trust the live field and open an issue with the date, account type, and screenshot when possible.
