# Templates

Practical structures you can copy, fill, and adapt without installing anything.

## Available templates

### [LinkedIn Post Templates](linkedin-post-templates.md)

A swipe file of 24 post structures for founder-led LinkedIn content.

Quick start:

1. Choose the template that matches the job of the post.
2. Replace every placeholder with a real detail.
3. Keep one idea per post.
4. Remove any line that does not add context, tension, evidence, or value.
5. Format and preview the final draft with the [PromiseClick LinkedIn Post Formatter](https://www.promiseclick.com/linkedin-free-tools/linkedin-post-formatter).

Starter request for an AI assistant:

```text
Use template [NUMBER OR NAME] from this swipe file to structure my idea. Preserve my facts and voice. Do not invent examples, metrics, or customer results.

Idea: [PASTE YOUR IDEA]
Audience: [YOUR ICP]
Goal: [AUTHORITY, CONVERSATION, PROOF, OR RESOURCE]
```

## Template standards

- A template controls structure, not substance.
- Replace placeholders with specific language before publishing.
- Do not force an idea into a template that changes its meaning.
- Keep proof traceable to the source.
