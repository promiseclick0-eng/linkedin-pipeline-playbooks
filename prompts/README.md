# Prompt Packs

Copy-and-use prompts for people who want a focused workflow without installing a full skill.

## Available prompt packs

### [Founder Hooks Prompt Pack](founder-hooks.md)

Use it to create several LinkedIn hook directions from one real idea, story, lesson, result, or point of view.

Quick start:

1. Open the prompt pack.
2. Copy the master prompt.
3. Replace every bracketed field with real context.
4. Run it in Claude, ChatGPT, or another capable assistant.
5. Choose the clearest angle, then edit it in your voice.

Starter request:

```text
Use this prompt pack to write 10 distinct hooks for the idea below. Keep the claims accurate, avoid generic curiosity bait, and label the angle behind each hook.

Idea: [PASTE YOUR IDEA]
Audience: [YOUR ICP]
Point of view: [WHAT YOU BELIEVE]
```

## Prompt-pack standards

- Replace placeholders before running a prompt.
- Provide real examples and evidence when the output includes claims.
- Do not publish generated copy without checking tone and accuracy.
- Prefer fewer strong outputs over many minor variations.
