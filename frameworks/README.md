# Frameworks

Practical operating systems that connect multiple tools and decisions without forcing one vendor or channel.

## Available frameworks

### [Clay-to-LinkedIn Prospecting Workflow](clay-to-linkedin-prospecting-workflow.md)

Use it to design an account-first workflow for:

- ICP and disqualifier definition
- Account sourcing and gating
- Conditional enrichment
- Explainable fit and confidence scoring
- LinkedIn-relevant signal review
- Warm action routing
- Human approval, cost controls, and learning

The framework intentionally excludes cold email and keeps no action as a valid route.

## Framework standards

- Define the decision before choosing tools.
- Preserve source evidence and unknown states.
- Keep automation reviewable by a human owner.
- Separate platform facts from operating recommendations.
- Include stop conditions, quality checks, and a learning loop.
